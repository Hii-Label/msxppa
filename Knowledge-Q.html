Rulles Perhitungan Harga

1. Dapat Spesifikasi
2. ERP -> Perhitungan Harga
3. Verifikasi
4. Ajukan Permintaan Harga Ke Owner
5. Buat Penawaran Harga
6. Share Ke Customer

Syarat Permintaan Harga -> Customer (-15 Mnt)

- Nama Customer ( harus ada Wajib)
- Nama Item ( harus ada Wajib, bisa Lihat Design atau Pikir Sendiri)
- Design Gambar ( Harus Menyertakan, Wajib ( Jika Bukan Blank) )

1. Ukuran / Size Item ( Wajib Dilampirkan Di Awal)
2. Bahan Baku / Raw Material (Wajib Ditanhyakan Di Awal)
3. Jumlah Pesanan / Qty Order ( Wajib Ditanyakan Di Awal)
4. Hasil Jadi Item ( Tidak Wajib Tapi Harus Ditanyakan)
5. Finishing Item ( Harus Di rekomendasikan)
Note :
Mohon untuk Customer Service Kondisikan, Jika Tidak Ada atau Spesifikasi Tidak Lengkap
